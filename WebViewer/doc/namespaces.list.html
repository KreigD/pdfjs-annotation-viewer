displayModeChanged

A jQuery event bound on the element, triggered when the display mode has changed.

Deprecated:
  • Since version 1.7. Use layoutModeChanged instead.
Example
$('#viewer').bind('displayModeChanged', function(event, data) {
    //event triggered
});
